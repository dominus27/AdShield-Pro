PRIVACY POLICY - ADSHIELD PRO
Last Updated: December 24, 2025

1. INTRODUCTION
AdShield Pro ("we", "our", "the extension") is committed to protecting your privacy. This policy explains how we handle your data.

2. DATA COLLECTION
AdShield Pro does NOT collect, store, or transmit any of your personal data or browsing information.

Specifically, we DO NOT collect:
- Browsing history
- Website visits
- Search queries
- Personal information
- IP addresses
- Location data
- Cookies or tracking data
- Any identifiable information

3. DATA STORAGE
The extension stores the following data LOCALLY on your device only:
- Extension settings (blocking preferences)
- Statistics (ads blocked count - stored locally)
- Premium license key (if activated - stored locally in Chrome sync storage)

This data:
- Never leaves your device
- Is not transmitted to our servers
- Is not shared with third parties
- Can be deleted by uninstalling the extension

4. PREMIUM LICENSE VALIDATION
If you activate Premium, the extension validates your license key with Gumroad's API by sending:
- Your license key (only)
- Product ID (AdShield Pro's identifier)

This validation:
- Happens only when you activate Premium
- Does not send any personal browsing data
- Uses Gumroad's secure API (https://api.gumroad.com)
- Stores the result locally on your device

Gumroad's privacy policy: https://gumroad.com/privacy

5. PERMISSIONS USED
The extension requires certain Chrome permissions to function:

- "declarativeNetRequest" - Block ads and trackers using filter rules
- "storage" - Save your settings locally
- "tabs" - Update badge counter with blocked ads count
- "webNavigation" - Detect page loads for statistics
- "contextMenus" - Add right-click menu options
- "<all_urls>" - Apply ad blocking on all websites you visit

These permissions do NOT allow us to see or collect your browsing data.

6. THIRD-PARTY SERVICES
The extension does not use any third-party analytics, tracking, or data collection services.

The only external connection is:
- Gumroad API (only during Premium license validation)

7. CONTENT BLOCKING
The extension blocks ads and trackers by:
- Using declarativeNetRequest rules (processed by Chrome)
- Running content scripts on web pages (locally)
- Never sending your browsing data anywhere

8. CHANGES TO POLICY
We may update this policy. Changes will be posted on:
- Chrome Web Store listing
- Extension options page

10. YOUR RIGHTS
You have the right to:
- Use the extension without providing any personal data
- Delete all extension data by uninstalling
- Request information about data practices (we don't collect any)

SUMMARY: AdShield Pro is privacy-first. We don't collect, store, or share any of your personal data or browsing information. Everything happens locally on your device.
